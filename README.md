# Gravity Forms WooCommerce Integration

A lightweight WordPress plugin that connects **Gravity Forms** to **WooCommerce** and dynamically:

- Adds a WooCommerce product to the cart on form submission  
- Applies dynamic pricing  
- Inserts event/ticket information into the product title  
- Lets the admin configure all field IDs from the WordPress dashboard  
- No meta duplication — clean cart UI  

## Features
- Map multiple Gravity Forms to one WooCommerce product
- Dynamic event name, ticket type, and ticket count
- Dynamic pricing per ticket
- Admin settings page under: **WooCommerce → GF Woo Integration**
- No code editing needed — fully configurable in backend

## Installation
1. Download or clone this repository.
2. Upload the folder to `/wp-content/plugins/`.
3. Activate the plugin in WordPress.

## Settings Page
Go to:

WooCommerce → GF Woo Integration


Configure:
- Allowed form IDs
- Reservation product ID
- Ticket field IDs (type, price, count)
